Maybe you didn't know that...

…the maximum possible score playing Pacman (by eating every possible dot, power pellet, fruit, and enemy) without losing a single life is 3,333,360 points.
The first person to achieve it was Billy Mitchell on July 3, 1999 in a 6-hour game.

…pacman was designed to be a no-ending game (so as long a life is available, a player may continue the game indefinitely) but at 255-th level, due to a rendering bug, too many fruits appear on the bottom line, causing a kill-split-screen to appear. So it's impossible to go further into the game.

…the name of the blue ghost in the american version of Pacman is "Inky" but its original japanese name is actually "Kimagure" (気まぐれ).

This demo uses only CSS3 and it is absolutely javascript-free. It was specifically designed for Firefox only: on Opera it looks quite good and on Webkit and IE9 some animations fail — Some content is also available if CSS are disabled/not supported.

Animations, transformations, content repositioning, score update and interactive clickable (blinking) objects are all managed only through mediaqueries.
